# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-21

### Added
- Initial release of AI CV Builder
- JSON-based CV input format with schema validation
- GitHub README integration for public and private repositories
- OpenRouter integration with google/gemini-3-flash-preview model
- High-reasoning AI prompt engineering for award-winning CVs
- Interactive CLI for selecting repositories on-the-fly
- DOCX to JSON conversion tool for existing CVs
- One-page, ATS-optimized DOCX generation with 0.25" margins
- Compact formatting with tight spacing and minimal content loss
- ATS keyword injection and metric-driven bullet point generation
- Support for profiles, experience, projects, education, skills, certifications
- README caching to avoid repeated GitHub API calls
- Comprehensive error handling and logging

### Features
- Convert existing DOCX CVs to JSON format
- Download READMEs from GitHub (public and private repos)
- AI-powered CV tailoring with metrics and impact statements
- One-page professional formatting
- Section headers with horizontal lines
- Skill groups and organized sections
- Contact information with multiple links
